Set Score Scatterplot

This visualization depicts 4783 Sets from Matches charted by the Match Charting Project. Hover over an indidual set to see the opponents; click a set to see the other sets that are part of the same match. Hover over the legend to filter the sets by set score; click a set score on the legend to "zoom" in on a collection of sets. The plot can be reorganized by toggling the radio buttons in the lower right corner. Filter by Player (lower right).
